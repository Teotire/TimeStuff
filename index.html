BlockWalker

Data pack for the Minecraft versions 1.17+

This is a data pack where the worlds blocks turn into their mineral block equivalent when you're close enough to do so. It's a pretty satisfying experiance to see the world change with this pack!

I have not added every block in the game.

Instructions

For this data pack to work you need to:

  • Start your game with cheats enabled
  • Start your game in the creative gamemode
  • Run the command "/reload"
/reload

For best experience set the gamerule sendCommandFeedback to false.
/gamerule sendCommandFeedback false

NPCs in Vanilla Minecraft

Click here to download NPCs in vanilla Minecraft combo pack!

NPC texture pack and data pack for the Minecraft version 1.18

This is a data pack and texture pack for NPCs in vanilla Minecraft. They're interactive, so if you right-click them they will say something. You can edit the files and reuse them for your own projects IF you credit me.

Instructions

File-wise:

  1. Download the NPC_files.zip.
  2. Extract the resource and data packs
  3. Place the resource pack in your resourcepacks folder
  4. Place the data pack in your datapacks folder of your chosen world

In-game:

  1. Run the command "/function summon derpa" at the location of your choosing.
  2. If that doesn't work, run the "/reload" command

/function summon derpa

/reload

For best experience set the gamerule sendCommandFeedback to false.
/gamerule sendCommandFeedback false
