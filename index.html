BlockWalker

BlockWalker

Click here to download BlockWalker!

Data pack for the Minecraft versions 1.17+

This is a data pack where the worlds blocks turn into their mineral block equivalent when you're close enough to do so. It's a pretty satisfying experiance to see the world change with this pack!

I have not added every block in the game.

Instructions

For this data pack to work you need to:

  • Start your game with cheats enabled.
  • Start your game in the creative gamemode.
  • Run the command "/reload".
/reload

For best experience set the gamerule sendCommandFeedback to false.
/gamerule sendCommandFeedback false

NPCs v3.0

NPCs in Vanilla Minecraft

Click here to download NPCs in vanilla Minecraft combo pack!

NPC texture pack and data pack for the Minecraft version 1.19+. Might work on earlier versions as well. I haven't tested this.

BUGS: Peasant NPCs are really bugged. I'm working on it. I wouldn't recomend using them.

NPCs will sometimes turn invisible. To fix this all players need to distance themselves at least 20 blocks from the NPCs, then approach them again.

This is a data pack and texture pack for NPCs in vanilla Minecraft. They're interactive, so if you right-click them they will say something. You can edit the files and reuse them for your own projects, but don't take my work as your own.

If you want to create your own NPC, but can't figure it out for yourself; message me and I'll help you out with that!

Instructions

File-wise

  1. Download the NPC_files.zip.
  2. Extract the resource and data packs.
  3. Place the resource pack in your resourcepacks folder.
  4. Place the data pack in your datapacks folder of your chosen world.

In-game

  1. Run the command "/function summon:derpa" on the location of your choosing.
  2. If it doesn't work run the "/reload" command and try the previous step again.
  3. There's an interactive documentation in-game you can acess using the command "/function help:main".

/function summon:derpa

/reload

/function help:main

For best experience set the gamerule sendCommandFeedback to false.
/gamerule sendCommandFeedback false

Morality Meter

Morality Meter in Vanilla Minecraft

Click here to download the Morality Meter combo pack!

Morality texture pack and data pack for the Minecraft version 1.18

BUGS: This data pack only works in single-player.

This is a data pack and texture pack for the Morality Meter. Some things you do will affect the morality meter. Good actions will make it go up, and evil actions will make it go down.

Instructions

File-wise

  1. Download the Morality_files.zip.
  2. Extract the resource and data packs.
  3. Place the resource pack in your resourcepacks folder.
  4. Place the data pack in your datapacks folder of your chosen world.

In-game

  1. The Datack should work directly.
  2. If it doesn't work run the "/reload" command and try the previous step again.
  3. (Optional) Run the "/function tools:signs" to get access to some testing tools.

/function general:setup

/reload

/function tools:signs

For best experience set the gamerule sendCommandFeedback to false.
/gamerule sendCommandFeedback false
